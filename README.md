# Test d'API REST via Restler
Projet pour tester l'outil ***Restler***.
## Commandes possibles en GET: (à saisir après index.php)
* /say/hello
* /say/hi
* /animals/listall
* /animals/listsomeanimals
* /restricted?key=password
* /authors

* PUT /authors/{id}
